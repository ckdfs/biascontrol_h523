# 关键不变量

以下约束是系统正确性的基础，修改前必须充分理解其物理含义。

1. **Goertzel 块长 N=1280**：在 64 kSPS 下对应恰好 20 个整数导频周期（20 ms），确保零频谱泄漏。改变采样率或导频频率时，N 必须同步调整。

2. **H2 使用 Q 分量**：H2 信号取 `h2_mag × sin(h2_phase)`，而非 `cos(h2_phase)`。硬件处理延迟使 H2 相位偏移约 82°，I 分量（cos）信号约弱 7×，且导致仿射矩阵近奇异。pass2 扫描与运行时均一致使用 sin。

3. **仿射矩阵行按 Bessel 比率缩放**：运行时有效矩阵 `M_eff` 的行分别乘以 `J₁(m_now)/J₁(m_cal)` 和 `J₂(m_now)/J₂(m_cal)`，再求逆，以补偿调制深度变化。

4. **归一化到单位圆**：`[obs_x, obs_y]` 归一化后相位估计与光功率和 RF 功率无关；跳过此步将引入幅度耦合误差。

5. **DAC 输出 = bias_setpoint + pilot_sample，钳位到 ±10 V**：导频叠加在 DAC 输出侧完成，减法器电路将 DAC 单极输出映射到 ±10 V 双极偏压。

6. **PI 具有积分抗饱和钳位；积分初值从粗扫获取**：启动时从 `start` 命令的粗扫结果播种积分器，避免冷启动超调。

7. **DRDY 中断具有最高 NVIC 优先级**：控制环路在更低优先级运行；禁止在控制路径中调用任何阻塞函数（见 `docs/lessons.md` § ISR 隔离）。

8. **DC 通道（CH1）仅用于监控**：CH1 DC 均值不参与运行时误差计算或锁定判据，防止直流漂移干扰相位观测。
